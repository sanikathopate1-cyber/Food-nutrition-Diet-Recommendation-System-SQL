# Food-nutrition-Diet-Recommendation-System-SQL
Food Nutrition & Diet Recommendation System | SQL
Designed and implemented a Food Nutrition & Diet Recommendation System using SQL to manage and analyze user, food, and dietary data.
Developed complex SQL queries using JOINs, subqueries, GROUP BY, and HAVING to analyze nutritional values, calorie intake, and food categories.
Extracted insights to identify diet patterns, high-calorie foods, and personalized nutrition recommendations
